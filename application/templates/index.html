Inventory Management

This is inventory management system, where users can manage their inventory. This system provides following functionality:

    Creating user
    Signing in
    Creating an inventory
    Removin inventory
    Add products to inventory
    Add Orders to product: incoming and outgoing
    Adding safetystock limits for products
    Monitor for products, where current stock level is under safety stock
    Charts about individual product and summary pie chart
    System provides ability to contribute to other users inventories
